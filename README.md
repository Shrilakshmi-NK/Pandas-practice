# Pandas-practice

A comprehensive learning repository focused on mastering **Pandas**, a powerful Python data manipulation and analysis library. This repository contains Jupyter notebooks covering essential Pandas concepts with practical examples and hands-on exercises.

## 📚 Contents

### Jupyter Notebooks

1. **Reading in Files.ipynb**
   - Learn how to read data from various file formats (CSV, Excel, JSON, TXT)
   - Importing data from different sources into Pandas DataFrames

2. **Indexing.ipynb**
   - Master indexing techniques in Pandas
   - Access and select data using different indexing methods
   - Label-based and position-based indexing

3. **Filtering and Ordering.ipynb**
   - Filter data based on conditions
   - Sort and arrange data in various ways
   - Boolean indexing and query operations

4. **Data Cleaning.ipynb**
   - Handle missing values and duplicates
   - Data validation and transformation
   - Preparing raw data for analysis

5. **Group by and Aggregating.ipynb**
   - Grouping data by categories
   - Aggregation functions (sum, mean, count, etc.)
   - Computing statistics on grouped data

6. **Merge, Join, and Concatenate.ipynb**
   - Combining multiple DataFrames
   - Different join types (inner, outer, left, right)
   - Concatenating data along rows and columns

7. **EDA in Pandas.ipynb**
   - Exploratory Data Analysis techniques
   - Statistical summaries and distributions
   - Understanding data patterns and relationships

8. **Pandas Visualization.ipynb**
   - Creating visualizations with Pandas
   - Plotting techniques for data exploration
   - Graphical representation of data

### Sample Datasets

- **Customer Call List.xlsx** - Customer contact information dataset
- **Flavors.csv** - Ice cream flavor ratings data
- **Ice Cream Ratings.csv** - Customer ratings for ice cream flavors
- **LOTR.csv** & **LOTR 2.csv** - Lord of the Rings related data
- **countries of the world.csv** / **.txt** - Global country statistics
- **world_population.csv** & **world_population_excel_workbook.xlsx** - World population data
- **json_sample.json** - Sample JSON file for data import practice
